# StratDistill Extended Analysis

Generated by `scripts/update_all.py`.

## Scope

- Discovery source: `https://stats-data.hyperliquid.xyz/Mainnet/vaults`
- Detail enrichment source: `POST https://api.hyperliquid.xyz/info` with `type=vaultDetails`
- This run enriched up to top-250 strategies by base composite score.

## Outputs

- `data/processed/strategy_registry.csv` (detail registry)
- `data/processed/strategy_ranked_extended.csv` (extended ranking table)
- `reports/figures/top20_extended_score.png`
- `reports/figures/apr_vs_drawdown_tvl.png`
- `reports/figures/tvl_distribution_log10.png`
- `reports/figures/feature_missingness_ratio.png`

## Extended Ranking Logic

`extended_score = 0.75 * composite_score + 0.08 * score_commission + 0.07 * score_withdrawable + 0.10 * score_av_depth`

Where:
- `score_commission`: lower leader commission is better
- `score_withdrawable`: higher maxWithdrawable / maxDistributable is better
- `score_av_depth`: deeper all-time account value history is better

## Notes

- Public data has uneven coverage on some dimensions (followers, some timeline depth).
- We use dynamic handling to avoid over-weighting sparse fields.
